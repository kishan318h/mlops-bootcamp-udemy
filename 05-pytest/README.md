## 1. Install library
- Syntax: `pip install pytest`

## 2. Test file creation
- Create file named `test_*.py` or `*_test.py`

## 3. Test function naming
- Start function name with test_

## 4. Assertion
- Use Python's built-in asset statement

## 5. Class naming
- Start class names with **Test\***

## 6. Method naming
- Start method names with **test_**
