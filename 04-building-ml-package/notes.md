## Create MANIFEST.in file
- This file contains list of instructions, one per line, instructing setuptools to add or remove some set of files from the sdist (source distribution)

## Creating VERSION file
- Create a version file with name 'VERSION' and save the version number
- in __init__.py in prediction model save the version number in __version__
- in doing so we can check the model version number using `model_name __version__`
